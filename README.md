# Godwoken Bridge

An asset bridge for `CKB <=> Godwoken`

## Accessing the Godwoken Bridge

- mainnet: https://mainnet.bridge.godwoken.io
- testnet: https://testnet.bridge.godwoken.io

## Development

- NodeJS >= 14
- Yarn >= 1.22

```sh
yarn install
yarn test
yarn start
```
