# Light Godwoken

This is a demo UI for depositing and withdrawing assets to Godwoken, which is a CKB Layer 2 chain to CKB chain.
You will need to have MetaMask installed to use this demo.

## Quick Start

- MetaMask
- NodeJS >= 14

```sh
yarn install
yarn start
```
